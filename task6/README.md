# Java GUI - ToDo App

## Objective
Build a simple To-Do List application using Java Swing.

## Features
- Add new tasks
- Delete selected tasks
- Scrollable task list
- GUI built using JFrame, JButton, JTextField
- Event handling using ActionListener

## Tools Used
- Java
- Swing (built-in)
- IntelliJ IDEA Community Edition

## Concepts Used
- Swing components
- Event handling
- Layout managers (BorderLayout)
- JScrollPane
