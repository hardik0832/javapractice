# Student Record Management System

## Objective
CLI-based CRUD system to manage student records using Java.

## Features
- Add Student
- View Students
- Update Student
- Delete Student

## Concepts Used
- Classes & Objects
- Encapsulation
- ArrayList
- Loops & Switch Case
- CRUD Operations

## How to Run
1. Compile:
   javac Student.java StudentManagementSystem.java

2. Run:
   java StudentManagementSystem
