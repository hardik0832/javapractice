# Bank Account Simulation (Java OOP)

## Objective
Simulate basic bank operations using Java OOP concepts.

## Features
- Deposit and Withdraw money
- Maintain transaction history
- Display account balance
- Savings account with interest
- Method overriding (withdraw)
- Inheritance used

## Concepts Used
- Classes & Objects
- Inheritance
- Method Overriding
- Encapsulation
- Constructor

## How to Run
1. Compile all files:
   javac *.java
2. Run:
   java BankSimulation
